## ✨ About The Project

INFINITUS 2026 is a breathtaking, highly interactive web experience designed to showcase an upcoming mega-event. Leveraging cutting-edge web technologies, it provides a seamless, immersive cosmic journey right from the browser. 

The application uses **React Three Fiber** for stunning 3D visual effects (like the interactive Star Canvas) and **Framer Motion** for butter-smooth scroll animations, delivering a premium "wow" factor.

### 🚀 Key Features

- **Immersive 3D Graphics**: Interactive particle systems and canvases built with Three.js.
- **Scroll-Linked Animations**: Dynamic components like Zoom Parallax powered by Framer Motion.
- **Modern UI/UX**: Custom flare cursors, glassmorphism elements, and a dark space-themed aesthetic (`#030014`).
- **Fully Responsive**: Flawless experience across desktop, tablet, and mobile screens.
- **Performant Next.js App Router**: Optimized routing, SEO-ready pages, and blazing-fast loading speeds.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **3D Rendering:** [Three.js](https://threejs.org/) + [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) + [Drei](https://github.com/pmndrs/drei)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) + custom components
- **Icons:** [Tabler Icons](https://tabler-icons.io/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment:** [Vercel](https://vercel.com/)

## 💻 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher)
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository** (if applicable):
   ```bash
   git clone <repository-url>
   cd infinitus-2026
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or yarn install / pnpm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   # or yarn dev / pnpm dev
   ```

4. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the result.
